## 1. 产品概述

多设备充电线管理系统——面向数码爱好者与设备管理员，解决充电线型号繁多、适配混乱、寿命难以追踪、丢失损坏无记录等痛点，实现充电线全生命周期数字化管理。

- 目标用户：拥有多种电子设备的个人用户、IT设备管理员
- 核心价值：减少充电线重复购买、避免不匹配浪费、提前预警寿命到期

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 账号密码注册登录 | 管理自己的充电线数据 |

### 2.2 功能模块

1. **登录页**：JWT 认证登录/注册
2. **仪表盘首页**：数据概览、到期提醒、月度消耗统计
3. **充电线管理页**：充电线列表 CRUD、多条件筛选、图片上传
4. **充电线详情页**：完整信息展示、适配设备、寿命预估、状态变更

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 登录页 | 登录表单 | 账号密码登录、注册切换、JWT Token 获取 |
| 仪表盘 | 统计卡片 | 充电线总数、正常/损坏/丢失数量、即将到期数量 |
| 仪表盘 | 月度消耗图表 | 近6个月充电线新增/损坏/丢失趋势柱状图 |
| 仪表盘 | 到期提醒列表 | 寿命即将到期的充电线列表，支持一键标记 |
| 充电线管理 | 筛选栏 | 按接口类型、长度、颜色、状态、购买日期范围筛选 |
| 充电线管理 | 充电线列表 | 表格展示所有充电线，支持排序与分页 |
| 充电线管理 | 新增/编辑弹窗 | 录入型号、接口类型、适配设备、长度、颜色、购买日期、实拍图上传 |
| 充电线管理 | 状态标记 | 标记为正常/损坏/丢失，记录变更时间 |
| 充电线详情 | 详情面板 | 展示充电线全部信息、实拍图预览、适配设备列表 |
| 充电线详情 | 寿命进度条 | 基于购买日期预估剩余寿命，可视化展示 |

## 3. 核心流程

用户注册登录 → 进入仪表盘查看概览与提醒 → 新增充电线录入信息与图片 → 在管理页筛选浏览 → 查看详情与寿命预估 → 标记损坏/丢失 → 月度统计自动更新

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深空灰(#1a1a2e, #16213e) + 电光蓝(#0fbcf9, #00d2ff) + 纯白(#ffffff)
- **辅助色**：状态绿(#00e676)、警告橙(#ff9100)、危险红(#ff1744)
- **按钮风格**：圆角微凸，电光蓝主按钮、深空灰次按钮
- **字体**：标题使用 Rajdhani（科技感），正文使用 Noto Sans SC
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式布局
- **图标风格**：数据线、充电头简约线条图标，基于 Lucide Icons
- **背景**：深色渐变底 + 微弱电路纹理/网格线

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录页 | 登录表单 | 居中卡片、深色背景、电光蓝输入框聚焦、线条充电图标装饰 |
| 仪表盘 | 统计卡片 | 4列网格、深灰卡片、电光蓝数字高亮、hover微动 |
| 仪表盘 | 月度消耗图表 | 柱状图、电光蓝/灰蓝双色柱体、深色背景 |
| 仪表盘 | 到期提醒 | 列表卡片、警告橙色标识、进度条渐变 |
| 充电线管理 | 筛选栏 | 水平排列下拉框、电光蓝边框、深灰底色 |
| 充电线管理 | 充电线列表 | 深色表格、电光蓝行hover、状态标签彩色 |
| 充电线管理 | 新增/编辑弹窗 | 深色弹窗、电光蓝强调、图片上传区域虚线边框 |
| 充电线详情 | 详情面板 | 左右分栏、图片预览、寿命圆环进度 |

### 4.3 响应式

- 桌面优先设计（1440px 基准）
- 平板适配（768px+）：侧边栏折叠为图标
- 移动端适配（<768px）：单列布局、底部导航
